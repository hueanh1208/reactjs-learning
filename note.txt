1/ cái đặt project:
    npx create-react-app ten-app
2/ chạy project:
    2.1 mở terminal ngay tại package.json
    2.2 gõ lệnh: npm run start
3/ tài liệu về reactjs https://reactjs.org/docs/getting-started.html
4/ flow code reactjs: index.html => index.js => App.js
5/  reactJS sẽ ko hiểu html, mà nó chỉ hiểu jsx( class => className)
6/ Lý do dùng component là:
    - dễ quản lí (chia để trị)
    - tái sử dụng (ưu tiên hàng đầu)
7/ Điều kiện để tạo ra 1 component:
    - tái sử dụng (ưu tiên hàng đầu)
    - dễ quản lí (chia để trị)
8/ Có 2 dạng component:
    - class
    - function 
bonus: 
1. 1 website thường có 2 dạng:
    - single page application (chuyển trang ko bị load)
    - multiple page application(chuyển trang bị load) 
2. về SEO thì  multiple page application tốt hơn single page application
3. reactjs(single page application) thì ko SEO tốt nhưng Nextjs giúp reactjs SEO tốt hơn
